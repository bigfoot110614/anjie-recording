定位布局 { 相对定位--没有脱离文档流，参照物是自己本身，不会影响其他元素的位置，没有改变行内元素的属性，但是可以设置上下左右的值，层级也发生了变化。 绝对定位--脱离了文档流，在没有设置参照物的情况下，参照物是可视窗口，人为设置父级元素为参照物，改变了行内元素自身的属性，可以设置行内元素的宽度和高度，padding值和margin值。所有定位的元素都会找一个起点，并且层级发生变化，可以用z-index设置层级的值。（只要父级元素有三种定位中任意一种，就可以成为绝对定位元素的参照物） 固定定位--脱离了文档流，参照物永远是可视窗口，改变了行内元素自身的属性，可以设置行内元素的宽度和高度，padding值和margin值。 z-index只能作用于兄弟元素，值不用太大，默认为0。 }
